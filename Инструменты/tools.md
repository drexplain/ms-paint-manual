# Инструменты

В MS Paint доступны следующие инструменты:

- **Карандаш** и **кисть**: для рисования линий. Кисть может быть разной толщины.
- **Ластик**: стирает нарисованное.
- **Фигуры**: для рисования прямоугольников и кругов. Чтобы получить ровный квадрат или круг, удерживайте клавишу Shift.
- **Заливка**: ведро, закрашивающее замкнутую область одним цветом.
- **Текст**: кнопка с буквой А для добавления текста.
- **Лупа**: увеличивает изображение для рисования мелких деталей.
- **Выделение**: позволяет вырезать, копировать или перемещать части изображения.
